## Setup

```sh
# Install dependencies
npm install

# Compile sources
npm run build

# Run tests
npm run tests

# Run the bot
npm start
```

## Docker

```sh
# 1. Build container
docker build -t ultimate-probot .

# 2. Start container
docker run -e APP_ID=<app-id> -e PRIVATE_KEY=<pem-value> ultimate-probot
```
